Data Searches
=============

DataSearches tool for ArcGIS Pro

Copyright © 2024 Andy Foy Consulting

Overview
--------
DataSearches is an ArcGIS Pro tool which lists a set of available GIS layers and allows the user to select which layers to extract for the current data search, specifying a radius from feature(s) of interest. The output files are created as ESRI Shapefiles (.shp) files, Comma Separated Value (.csv) files or Text (.txt) files.

Requirements
------------
The DataSearches tool requires the following:

 - ArcGIS Pro 3.3 or higher

Other versions of the tool are available for:

 - ArcGIS 10.1 or higher
 - MapInfo 32bit or 64bit
